mAItion can provide answers with citations to source chunks used during retrieval.

## How It Works

- Retrieval finds relevant indexed chunks
- Answer generation uses retrieved context
- Response includes source references for verification

## Why It Matters

- Transparent answer provenance
- Easier fact-checking by users
- Better trust in AI responses
